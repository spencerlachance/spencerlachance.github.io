Simple Animator

For my Object-Oriented Design class, a classmate and I implemented an application which reads input from a text file containg a description of an animation, and outputs the animation in one of four ways:
  • export a (more readable) text description of the animation
  • export SVG code capable of rendering the animation
  • play the animation in a simple window
  • play the animation in a more interactive window with various controls
