Simple Animator

 

 

For my Object-Oriented Design class, a classmate and I designed an application which reads input from a text file and outputs the animation it describes in one of four ways: by exporting a text description of the animation, exporting svg code capable of displaying the animation, playing the animation in a simple window, or playing the animation in a more interactive window with various controls.
